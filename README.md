# Sequenceanalyzer-app
[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://sequenceanalyzer-app.streamlit.app/)

Working
Protein Sequence Analyzer is a Streamlit web application implemented using Python programming language, along with other bioinformatics tools like Biopython, pandas, numpy, and matplotlib. The protein sequence is provided by the user. Using Biopython's ProteinAnalysis module, important physicochemical properties of the protein like molecular weight, isoelectric point, aromaticity, and instability index are calculated.
        Moreover, the program calculates the amino acid composition and represents it using bar charts drawn by matplotlib. The program further determines the percentage of acidic and basic amino acids including Aspartic Acid, Glutamic Acid, Lysine, Arginine, and Histidine, respectively. This information can be viewed in numeric format as well as using bar charts.
        Apart from that, the program offers features like prediction of phospho-acceptor sites, subsequence search (motifs), and obtaining structure details from the Protein Data Bank (PDB). In case PDB ID is provided, downloading and visualization of structure using 3D plotting by py3Dmol
